# Odin Orbital Station Restaurant Page

An orbital space station-themed restaurant webpage demonstrating dynamic JavaScript-driven content loading with Webpack, modular components, and responsive CSS styling to create an immersive online dining experience on an orbital station.

Referencing Odin's Project: [Restaurant Page](https://www.theodinproject.com/lessons/node-path-javascript-restaurant-page)

Live preview: [Odin's Orbital Station Restaurant Page](https://arsenlenaslov.github.io/odin-restaurant-page/)

## Features
- **Dynamic Tab Navigation**: Switch between Home, Menu, and Contact pages without reloading.
- **Modular Architecture**: Each page section (Home, Menu, Contact) is implemented as a separate JavaScript module.
- **Responsive Design**: The interface adapts to various screen sizes and devices.
- **CSS-in-JS Bundling**: Styles are bundled into the JavaScript for seamless loading.
- **Webpack Build**: for bundling and asset management.
- **GitHub Pages Deployment**: Deployed live using the gh-pages branch on GitHub.
